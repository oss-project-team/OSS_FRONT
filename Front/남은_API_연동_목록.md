# 남은 API 연동 목록

## 📊 현재 상태
- **연동 완료**: 18개 API
- **연동 필요**: 7개 API

---

## ❌ 아직 연동 안 된 API (7개)

### 1. 인증 관련 (3개)

#### `POST /api/v1/auth/reset-password` - 비밀번호 재설정
- **위치**: 비밀번호 찾기 페이지 (아직 없을 수 있음)
- **용도**: 이메일 인증 후 비밀번호 재설정
- **우선순위**: 낮음 (비밀번호 찾기 기능이 있다면 필요)

#### `POST /api/v1/auth/refresh` - 토큰 재발급
- **위치**: `utils/api.js` 또는 공통 유틸리티
- **용도**: Access Token 만료 시 자동으로 재발급
- **우선순위**: 중간 (사용자 경험 개선)

#### `POST /api/v1/auth/logout` - 로그아웃
- **위치**: `mypage.js` 또는 헤더/설정 페이지
- **용도**: 로그아웃 시 서버의 Refresh Token 삭제
- **우선순위**: 중간

---

### 2. 키워드 알림 관련 (3개)

#### `POST /api/v1/keywords` - 키워드 등록
- **위치**: 키워드 알림 페이지 (아직 없을 수 있음)
- **용도**: 관심 키워드 등록
- **우선순위**: 낮음

#### `GET /api/v1/keywords` - 내 키워드 목록
- **위치**: 키워드 알림 페이지
- **용도**: 등록한 키워드 목록 조회
- **우선순위**: 낮음

#### `DELETE /api/v1/keywords/<id>` - 키워드 삭제
- **위치**: 키워드 알림 페이지
- **용도**: 등록한 키워드 삭제
- **우선순위**: 낮음

---

### 3. 기타 (1개)

#### `GET /api/v1/messages/<id>` - 쪽지 상세 조회
- **위치**: `contact.js` (부분 연동됨)
- **용도**: 쪽지 상세 조회 및 읽음 처리
- **우선순위**: 낮음 (현재는 목록에서 모든 정보를 가져옴)
- **상태**: 쪽지 목록에서 이미 모든 정보를 가져오므로 별도 호출 불필요할 수 있음

---

## 🎯 우선순위별 정리

### 높은 우선순위
없음 (핵심 기능은 모두 연동 완료)

### 중간 우선순위
1. `POST /api/v1/auth/logout` - 로그아웃
2. `POST /api/v1/auth/refresh` - 토큰 재발급

### 낮은 우선순위
1. `POST /api/v1/auth/reset-password` - 비밀번호 재설정
2. 키워드 알림 관련 API 3개
3. `GET /api/v1/messages/<id>` - 쪽지 상세 조회
